SNEEditor by wongiseng

SNE Ontology Editor

SNE Editor developed to allow user to edit an instance of an existing OWL Ontology. This editor can be used to instantiate OWL classes, defines its data properties and configure its object properties. Form will be generated for filling data properties related to an OWL Classes. Object properties will be represented as input and output port. Input port of an OWL Class when the object property is has the OWL class as a domain, and Output port if the OWL Class is a domain of the object properties. Existing OWL Ontology needs to be first loaded into Sesame repository, all the forms generated based on sparql query performed into the sesame repository.

Dependencies

Google App Engine, WireIT Javascript Libraries , OWL API

Install

Pull, create GAE apps on eclipse, and deploy on your own GAE.
